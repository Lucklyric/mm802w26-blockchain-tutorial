# Assignment

## Setup

Requires **Python 3.11+** and [uv](https://docs.astral.sh/uv/).

```bash
cd assignment
uv sync
```

Edit `config.py` with your email, random numbers, and server URL.

## Exercises

Follow the tutorial at `http://<server-ip>/tutorial/`, steps 1–7.

| File | Exercise |
|------|----------|
| `part1_keygen.py` | Generate ECDSA key pair |
| `part3_explore.py` | Read blockchain state via API |
| `part4_create_block.py` | Construct a block |
| `part5_mine.py` | Mine with Proof of Work |
| `part6_submit.py` | Sign and submit block |
