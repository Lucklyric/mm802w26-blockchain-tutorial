Blockchain Tutorial — CMPUT 802

Welcome! In this hands-on assignment, you will build a Python client that interacts with a live blockchain server. By the end, you will have generated cryptographic keys, registered as a miner, constructed a block, mined it using Proof of Work, signed it, and submitted it to the chain.

What You'll Learn

  • SHA256 hashing — the building block of blockchain integrity
  • ECDSA key pairs — public-key cryptography on secp256k1 (Bitcoin's curve)
  • Proof of Work mining — brute-force nonce search for valid hashes
  • Digital signatures — proving ownership without revealing your private key
  • REST APIs — interacting with the blockchain via HTTP

Prerequisites

  • Python 3.11 or higher
  • uv package manager
  • A terminal / command line

Quick Setup

cd client
uv sync
# Edit config.py with your email and random numbers
uv run python part1_keygen.py
